# Panduan Penulisan Makalah Seminar Akuntansi — FEB UPB

Skill ini mengatur **cakupan terbatas** sesuai permintaan: struktur dokumen,
format daftar isi, sistem sitasi in-text & daftar pustaka, dan aturan teks miring
(italic) untuk kata asing. Panduan lain (jenis kertas, margin, nomor halaman,
tabel/gambar) mengikuti Pedoman Skripsi FEB UPB yang berlaku.

---

## 1. STRUKTUR MAKALAH SEMINAR AKUNTANSI

Urutan bagian makalah (dari depan ke belakang):

```
1. Cover / Halaman Sampul
2. Lembar Pengesahan / Halaman Persetujuan
3. Kata Pengantar
4. Daftar Isi
5. BAB I   — Pendahuluan
6. BAB II  — Tinjauan Pustaka
7. BAB III — Metode Penelitian
8. DAFTAR PUSTAKA
```

> **Catatan penting:** Makalah Seminar Akuntansi UPB **tidak** menggunakan
> BAB IV dan BAB V (berbeda dari skripsi penuh). Daftar Pustaka diletakkan
> langsung setelah BAB III, sebelum lampiran (jika ada).

---

## 2. FORMAT DAFTAR ISI

### Aturan Daftar Isi

Daftar isi memuat semua bagian utama dengan nomor halaman. Format standar:

```
KATA PENGANTAR ...........................................................  i
DAFTAR ISI .................................................................  ii
BAB I    PENDAHULUAN ......................................................  1
  1.1    Latar Belakang ...................................................  1
  1.2    Rumusan Masalah .................................................  X
  1.3    Tujuan Penelitian ...............................................  X
  1.4    Manfaat Penelitian ..............................................  X
BAB II   TINJAUAN PUSTAKA ................................................  X
  2.1    Landasan Teori ..................................................  X
  2.2    Penelitian Terdahulu ............................................  X
  2.3    Hipotesis Penelitian ............................................  X
  2.4    Kerangka Penelitian .............................................  X
BAB III  METODE PENELITIAN ..............................................  X
  3.1    Bentuk Penelitian ...............................................  X
  3.2    Populasi dan Sampel .............................................  X
  3.3    Variabel Penelitian .............................................  X
  3.4    Teknik Pengumpulan Data .........................................  X
  3.5    Teknik Pengolahan dan Analisis Data .............................  X
DAFTAR PUSTAKA ............................................................  X
```

### Aturan Penomoran Halaman

- **Bagian awal** (Kata Pengantar s.d. Daftar Isi): angka romawi kecil (i, ii, iii)
- **Bagian isi** (BAB I s.d. Daftar Pustaka): angka Arab (1, 2, 3, dst.)
- Nomor halaman ditempatkan di **kanan atas**, kecuali halaman yang diawali judul
  BAB → nomor di **bawah tengah**

## 3. SISTEM SITASI (IN-TEXT CITATION)

Makalah Seminar Akuntansi FEB UPB menggunakan **gaya APA** (American
Psychological Association). Kutipan dalam teks menggunakan format
**(Nama, Tahun)** atau **Nama (Tahun)**.

### Pola Dasar In-Text

| Kondisi | Format |
|---|---|
| Satu penulis | (Widyani, 2022) |
| Dua penulis | (Sari & Pravitasari, 2022) |
| Tiga penulis atau lebih | (Kurniawan et al., 2022) |
| Penulis disebutkan dalam kalimat | Menurut Lovita & Susanty (2021), ... |
| Kutipan langsung (ada nomor hal.) | (Terry, 2003, hlm. 45) |

### Aturan Penulisan *et al.*

- Gunakan *et al.* (miring) untuk 3 penulis atau lebih sejak sitasi pertama
- Contoh: *Budiadnyani et al., 2022* → **salah**; *(Budiadnyani, 2022)* → **benar**
  jika hanya 1 penulis; *(Kurniawan et al., 2022)* → **benar** jika ≥3 penulis

> **Perhatian:** *et al.* **selalu ditulis miring** karena merupakan singkatan
> Latin (*et alii* = dan kawan-kawan).

## 4. FORMAT DAFTAR PUSTAKA

Daftar pustaka disusun **alfabetis** berdasarkan nama belakang penulis pertama.
Nama belakang didahulukan. Gelar akademik **tidak** dicantumkan.

**Spasi pengetikan daftar pustaka:**
- Dalam satu entri: **1 spasi**
- Antar entri: **1,5 spasi**

### Contoh Format per Jenis Sumber

**a. Artikel Jurnal**
```
Nama Belakang, Inisial. (Tahun). Judul artikel. Nama Jurnal, Volume(Nomor),
    halaman-halaman.
```
Contoh:
```
Widyani, A. A. D., & Wijayani, D. R. (2022). Pengaruh e-commerce terhadap
    keputusan berwirausaha. Jurnal Akuntansi Indonesia, 11(2), 45–58.
```

**b. Buku Teks**
```
Nama Belakang, Inisial. (Tahun). Judul buku (edisi jika ada). Penerbit.
```
Contoh:
```
Hair, J. F., Black, W. C., Babin, B. J., & Anderson, R. E. (2014).
    Multivariate data analysis (7th ed.). Pearson Education.
```

**c. Artikel dari Website / Media Online**
```
Nama Penulis/Lembaga. (Tahun, Tanggal Bulan). Judul artikel. Nama Situs.
    URL
```
Contoh:
```
Reza, J. I. (2015, 10 Februari). Inikah dampak mematikan pemanasan global.
    Liputan6. http://tekno.liputan6.com/read/2304179/
```

**d. Skripsi / Tesis (tidak dipublikasikan)**
```
Nama Belakang, Inisial. (Tahun). Judul skripsi/tesis [Skripsi tidak
    dipublikasikan]. Nama Universitas.
```

**e. Lembaga sebagai Penulis**
```
Nama Lembaga. (Tahun). Judul publikasi. Nama Penerbit.
```
Contoh:
```
Badan Pusat Statistik. (2024). Statistik ketenagakerjaan Indonesia
    Februari 2024. BPS.
```

## 5. ATURAN TEKS MIRING (ITALIC)

Teks miring wajib digunakan dalam konteks-konteks berikut:

### 5.1 Kata dan Frasa Asing (Non-Indonesia)

Semua kata atau frasa berbahasa **asing** yang belum diserap ke dalam
Bahasa Indonesia baku wajib ditulis miring:

| Contoh benar | Keterangan |
|---|---|
| *e-commerce* | istilah teknologi Inggris |
| *decision making theory* | frasa teori bahasa Inggris |
| *business to business* (B2B) | jenis model bisnis |
| *peer to peer* | istilah teknologi |
| *client-server* | arsitektur sistem |
| *revenue system* | sub-sistem SIA |
| *et al.* | singkatan Latin |
| *stewardship* | konsep akuntansi |

**Pengecualian:** Kata serapan yang sudah masuk KBBI tidak perlu miring.
Contoh: manajemen, sistem, informasi, teknologi, analisis — **tidak miring**.

### 5.2 Istilah Teknis / Konsep yang Pertama Kali Diperkenalkan

Saat mendefinisikan sebuah istilah teknis baru, nama konsep tersebut dapat
ditulis miring pada **kemunculan pertama**:

```
Pengambilan keputusan (*decision making*) merupakan kemampuan individu
dalam membuat pilihan...
```

### 5.3 Judul Buku, Jurnal, dan Nama Kapal/Karya dalam Teks

Judul buku atau nama jurnal yang disebut di dalam teks ditulis miring:
```
Sebagaimana dikemukakan dalam *Multivariate Data Analysis* (Hair et al., 2014)...
```

### 5.4 Hal yang TIDAK Perlu Miring

- Nama orang, nama lembaga, nama negara
- Singkatan yang sudah umum: APA, BPS, OJK, TPT
- Angka dan simbol statistik (kecuali konvensi statistik tertentu seperti *n*, *p*)

## 6. PENULISAN SUBBAB DAN HEADING

Hierarki heading makalah:

```
BAB I  PENDAHULUAN              → HURUF KAPITAL SEMUA, tebal, di tengah
1.1    Latar Belakang           → Huruf Kapital Setiap Kata, tebal, kiri
1.1.1  Sub-subbab               → Huruf kapital hanya awal kalimat, tebal, kiri
1.1.1.1  Level 4                → Huruf kapital hanya awal kalimat, tebal-miring, kiri
```

## 7. TIPS PENERAPAN DALAM DOCX

Saat menghasilkan file `.docx` untuk makalah ini:

1. **Teks miring** → gunakan `run.italic = True` pada python-docx, atau tulis
   `*kata*` di markdown sebelum dikonversi
2. **Daftar isi** → susun manual dengan tab leader titik-titik, atau gunakan
   tabel tanpa border dengan kolom kiri (judul) dan kanan (nomor hal.)
3. **Daftar pustaka** → gunakan *hanging indent* (indentasi gantung):
   baris pertama rata kiri, baris selanjutnya menjorok 1,27 cm
4. **et al.** → selalu render sebagai teks miring

## 8. CHECKLIST SEBELUM FINALISASI

Sebelum menyerahkan makalah, periksa:

- [ ] Urutan bagian: Cover → Lembar Pengesahan → Kata Pengantar → Daftar Isi → BAB I → II → III → Daftar Pustaka
- [ ] Penomoran halaman: romawi kecil untuk bagian awal, arab untuk isi
- [ ] Semua sitasi in-text punya entri di Daftar Pustaka (dan sebaliknya)
- [ ] Semua kata asing ditulis miring
- [ ] *et al.* ditulis miring
- [ ] Format APA konsisten (tanda "&" untuk dua penulis, bukan "dan")
- [ ] Daftar Pustaka tersusun alfabetis
- [ ] Hanging indent pada setiap entri Daftar Pustaka
