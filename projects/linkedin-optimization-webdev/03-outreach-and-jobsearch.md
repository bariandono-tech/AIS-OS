# Job Search & Direct Outreach Strategy for Junior Vibe Coder

Dokumen ini memandu Anda dalam taktik berburu lowongan kerja remote lokal secara aktif di LinkedIn, serta cara melakukan *cold messaging* langsung ke pengambil keputusan (recruiter, tech lead, founder) untuk melompati seleksi CV konvensional.

---

## Part 1: Trik Pencarian Lowongan Remote Lokal di LinkedIn

Jangan hanya bergantung pada daftar lowongan di menu "Jobs" bawaan karena tingkat persaingan di sana sangat tinggi dan terkadang status "Remote" yang dipasang perusahaan adalah *clickbait* (sebenarnya hybrid/onsite). Gunakan taktik berikut:

### 1. Advanced Search Query di Menu "Jobs"
Gunakan operator Boolean untuk memperluas pencarian Anda. Masukkan ini di kotak pencarian posisi:
> `("React" OR "Next.js" OR "Frontend" OR "Fullstack") AND "Remote"`

*   **Lokasi**: Pilih `Indonesia`.
*   **Filter On-site/Remote**: Centang `Remote`.
*   **Date Posted (Tanggal Ditayangkan)**: Setel ke `Past 24 hours` atau `Past week`. Lowongan remote lokal junior cepat sekali ditutup karena pelamar yang membeludak. Melamar di 24 jam pertama memberikan peluang dibaca CV Anda hingga 5x lebih tinggi.

### 2. Berburu Lowongan via LinkedIn Feed (Pencarian Postingan)
Banyak startup kecil atau agensi web tidak memasang lowongan berbayar di LinkedIn Jobs. Mereka hanya menulis postingan biasa di feed mereka. 
Cari kata kunci ini di kolom pencarian utama LinkedIn, lalu ganti filter kategori menjadi **"Posts"** (bukan Jobs):
*   `"hiring remote react" indonesia`
*   `"mencari web developer remote"`
*   `"lowongan remote nextjs"`
*   `"open posisi frontend remote"`

Jika menemukan postingan dari Tech Lead atau Founder, langsung komentari dengan sopan dan kirimkan pesan koneksi (InMail).

---

## Part 2: Template Cold Outreach (Direct Pitching)

Melamar lewat tombol "Easy Apply" atau portal karir sering kali menyaring Anda secara otomatis jika sistem menggunakan filter gelar IT. Cara terbaik bagi Vibe Coder Non-IT adalah **menghubungi pengambil keputusan secara langsung**.

Cari profil orang yang bekerja di perusahaan target dengan jabatan:
*   *Recruiter / Talent Acquisition*
*   *Engineering Manager / Tech Lead / Lead Developer*
*   *Founder / Co-Founder / CTO* (untuk startup kecil di bawah 30 karyawan)

Kirimkan undangan pertemanan (Connection Request) dengan melampirkan pesan singkat (maksimal 300 karakter untuk koneksi gratis) atau kirim InMail/Chat jika sudah terhubung.

### Varian A: Pesan untuk Recruiter / HRD
> *Fokus: Kesopanan, kecocokan stack, dan akses instan ke portofolio proyek.*

**Pesan:**
> "Halo [Nama Recruiter], perkenalkan saya [Nama Anda]. Saya melihat [Nama Perusahaan] sedang membuka lowongan remote untuk posisi Junior Web Developer. Saya memiliki keahlian di stack React dan Next.js dengan fokus pada kecepatan build produk.
>
> Berikut adalah salah satu proyek web full-stack yang saya bangun dan deploy sendiri: [Link Live Web Portofolio]. Saya harap bisa berdiskusi lebih lanjut tentang bagaimana keahlian saya dapat berkontribusi di tim Anda. Terima kasih!"

---

### Varian B: Pesan untuk Tech Lead / Engineering Manager
> *Fokus: Pembicaraan teknis yang to-the-point, kebersihan kode (clean code), dan efisiensi.*

**Pesan:**
> "Halo Kak [Nama Tech Lead], salam kenal. Saya [Nama Anda], seorang web developer yang mendalami ekosistem JavaScript (React/Next.js/Node.js). Saya sangat menyukai pendekatan modern development workflows.
>
> Saya telah membangun proyek [Nama Proyek Utama Anda] dengan backend Express dan ORM Prisma yang bisa dicoba langsung di sini: [Link Live Web]. Kodenya juga saya kelola secara bersih di GitHub: [Link GitHub Repo].
>
> Jika tim Kakak sedang membutuhkan junior developer yang bisa onboard dengan cepat dan mandiri dalam eksekusi proyek remote, saya akan sangat senang untuk mengobrol lebih detail. Terima kasih!"

---

### Varian C: Pesan untuk Founder / CEO Startup Kecil atau Agensi
> *Fokus: Nilai bisnis (business value), solusi cepat untuk build MVP, fleksibilitas (bisa freelance/magang dahulu).*

**Pesan:**
> "Halo Pak/Bu [Nama Founder], salam kenal. Saya [Nama Anda], seorang Product-Oriented Web Developer spesialisasi React/Next.js. Saya melihat [Nama Perusahaan] sedang gencar mengembangkan platform digitalnya.
>
> Saya terbiasa membangun prototipe dan aplikasi web dengan cepat (fast shipping). Salah satu produk web yang saya rilis mandiri ada di: [Link Live Web].
>
> Saya sangat tertarik untuk membantu tim Anda secara remote, baik dalam skema full-time maupun project-based/freelance untuk mempercepat delivery platform Anda. Apakah ada waktu luang minggu ini untuk mengobrol singkat? Terima kasih!"

---

## Part 3: Golden Rules Outreach
1.  **Jangan Spam**: Kirimkan maksimal 5-7 pesan terkurasi setiap hari. Sesuaikan nama orang dan nama perusahaan di setiap pesan.
2.  **Siap dengan Penolakan**: Konversi cold outreach biasanya berkisar antara 10% - 20% (dari 10 orang yang Anda hubungi, mungkin hanya 1-2 yang membalas). Ini hal yang wajar. Jangan patah semangat.
3.  **Segera Follow-Up**: Jika mereka membalas dengan ketertarikan, langsung arahkan ke langkah konkret (misalnya menawarkan mengirimkan resume ATS-friendly atau menjadwalkan panggilan perkenalan singkat 15 menit via Google Meet).
