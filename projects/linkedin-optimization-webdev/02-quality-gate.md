# Quality Gate: LinkedIn Optimization for Vibe Coder

Gunakan checklist ini untuk memverifikasi bahwa profil LinkedIn Anda sudah 100% siap sebelum Anda mulai menekan tombol melamar kerja atau mengirim pesan outreach ke recruiter.

- [ ] **Headline**: Sudah mengandung keyword utama (React, Next.js, Node.js) dan TIDAK menulis kata-kata negatif/pasif ("Looking for job", "Non-IT degree").
- [ ] **About**: Sudah bercerita dengan gaya *storytelling* yang memposisikan diri sebagai *Product Developer* yang cepat mengeksekusi proyek. Tech stack tertulis dengan format list rapi.
- [ ] **Featured Section**: Sudah memajang minimal 2 proyek riil. Link live demo web aktif dan berfungsi (tidak rusak/error), serta link GitHub repository disertakan.
- [ ] **Experience**: Proyek mandiri sudah dideskripsikan sebagai pengalaman kerja nyata (misal: "Web Developer (Project-Based)") lengkap dengan pencapaian dan tech stack yang digunakan.
- [ ] **Skills**: Sudah menambahkan minimal 15 keahlian teknis (hard skills) spesifik.
- [ ] **GitHub Profile**: Repositori proyek portofolio diset ke "Public", README-nya rapi (dilengkapi screenshot/video demo web), dan file `.env.example` disediakan (tidak ada credential rahasia ter-upload).
- [ ] **Ready for Publish?** → [ ] PROCEED / [ ] BLOCK
