# 📊 Penghasilan Tidak Kena Pajak (PTKP)

PTKP adalah jumlah pendapatan batasan tertentu dari Wajib Pajak Orang Pribadi bulanan/tahunan yang dibebaskan dari pengenaan PPh Pasal 21.

## Tarif Tahunan PTKP Indonesia (Saat Ini)
*   **Wajib Pajak Sendiri (TK/0)**: Rp 54.000.000
*   **Tambahan WP Menikah (K/0)**: + Rp 4.500.000 (menjadi Rp 58.500.000)
*   **Tambahan Per Tanggungan (Maks 3)**: + Rp 4.500.000 per kepala keluarga (anak/anggota keluarga sedarah).

### Contoh Perhitungan Cepat
*   **K/1** (Kawin, 1 Anak):
    $$\text{PTKP} = \text{Rp } 54.000.000 \text{ (WP)} + \text{Rp } 4.500.000 \text{ (Kawin)} + \text{Rp } 4.500.000 \text{ (1 Anak)} = \text{Rp } 63.000.000$$
*   **K/3** (Kawin, 3 Anak):
    $$\text{PTKP} = \text{Rp } 54.000.000 \text{ (WP)} + \text{Rp } 4.500.000 \text{ (Kawin)} + \text{Rp } 13.500.000 \text{ (3 Anak)} = \text{Rp } 81.000.000$$
